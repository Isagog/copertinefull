Local dev setup procedure

0) Initial setup:
cd backend
poetry install
poetry run copertine-cli generate-secret  # Copy the output to .env
poetry run copertine-cli init-db  # Initialize the database
poetry run copertine-cli reset-db #  this is case the DB is there already but you want to clean it up

1) Start a mock SMTP server as 
docker-compose up mailpit -d (web interface will be on 8025)

2) Start the backend
cd backend && poetry run uvicorn src.main:app --reload

3) Start the frontend and go to the initial /copertine slug
cd frontend && pnpm dev
