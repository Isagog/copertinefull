Local dev setup procedure

Initial setup:
cd backend
poetry install
poetry run copertine generate-secret  # Copy the output to .env
poetry run copertine init-db  # Initialize the database

1) Start a mock SMTP server as 
python -m smtpd -c DebuggingServer -n localhost:1025

2) Start the backend
cd backend && poetry run uvicorn src.main:app --reload

3) Start the frontend and go to the initial /copertine slug
cd frontend && pnpm dev
